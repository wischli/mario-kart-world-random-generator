# Mario Kart World Random Track Generator

A web app that randomly selects 16 tracks for VS mode races in Mario Kart World. View your selection on an interactive world map and track your progress as you race!

**[Try it live](https://wischli.github.io/mario-kart-world-random-generator/)**

## Features

- **Random Selection**: Generate 16 random tracks from all 30 available courses
- **Interactive Map**: View selected tracks on a world map with numbered markers
- **Progress Tracking**: Mark tracks as completed and see which one is next
- **Shareable Links**: Share your track selection via URL
- **Dark/Light Mode**: Toggle between themes or use system preference
- **Mobile Friendly**: Works great on phones with touch-optimized controls
- **Offline Persistence**: Your progress is saved locally

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```
